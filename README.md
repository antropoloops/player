# Antropoloops player

https://play.antropoloops.com/

## Setup

- Clone repository
- `yarn install`
- `yarn start`

## Deploy

#### Staging

Just push to master: https://agitated-wright-68b7c5.netlify.com/

#### Production

 `yarn deploy`
